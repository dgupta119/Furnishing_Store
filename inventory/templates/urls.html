GET URLs:

  • articles/ - List and create articles
  • Retrieve a specific article by ID
  • List and create product configurations
  • Retrieve, update, and delete a specific product configuration
  • List all products which are available in inventory
  • Retrieve a specific product by ID

PUT Urls:

Use curl request to sell the product, to run this you can use CMD or Postman or any other API testing platform

  • curl -X "PUT" http://localhost:8000/inventory/products/15/sell/ -H "Content-Type: application/json"
